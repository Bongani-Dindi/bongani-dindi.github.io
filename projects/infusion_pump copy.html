Positive‑Displacement Infusion Pump (Veterinary)

Role: System design & analysis — architecture, rate/volume modes, pressure capability, and controls.

Overview

A compact infusion/dispense device for veterinary use. The pump delivers precise small‑volume doses and supports two operating modes: Direct Infusion (rate‑controlled) and Dispense (volume‑controlled). The design emphasizes accuracy at low flow, high back‑pressure tolerance, low power, and leakage resistance.

Objectives

  • Develop a positive‑displacement mechanism to ensure volumetric accuracy at small doses.
  • Implement rate control with closed‑loop feedback and volume batching with quick fill time.
  • Achieve high outlet pressure to handle varying viscosities/nozzles and tubing friction.
  • Minimize electrical power draw via friction reduction and lightweight drivetrain components.
  • Design for robustness, easy assembly, and high back‑pressure resistance before leakage.

Technical Work

Architecture. Mechanism options (peristaltic vs. piston/syringe). Chosen configuration is a positive‑displacement stage driven by a geared stepper/BLDC with encoder feedback. A flywheel/compensation strategy reduces pulsation and smooths load variations.

Control. Infusion mode uses a PID‑style controller on motor position/velocity with pressure‑proxy limits; Dispense mode runs a calibrated step count to a target volume. Calibration curves convert motor steps → displacement → delivered volume.

Sensing & Safety. Hall/optical encoders for position; optional inline pressure or motor current as back‑pressure proxy; occlusion detection, max‑pressure cutoff, and leak checks.

// Pseudocode: dual‑mode control
if (mode == INFUSION) {
    rate_cmd_ml_min = setpoint;
    target_steps_per_sec = k_rate * rate_cmd_ml_min;
    motor.pidStep(target_steps_per_sec, feedback_steps);
} else { // DISPENSE
    target_volume_ml = setpoint;
    target_steps = k_vol * target_volume_ml;
    motor.moveSteps(target_steps);
}
// Safety: stop if pressure_proxy > threshold or encoder slip detected

Results & Impact

  • Achieved accurate volumetric rate at low flow with compensation for friction/load changes.
  • Validated high back‑pressure capability across different nozzle diameters and viscosities.
  • Reduced power by minimizing drivetrain friction/mass; improved leakage resistance.
  • Modular architecture enables future disposable fluid path and battery operation.
